# Dialogue Splitter

Isolates voice in a video so you can hear it better. The other audio is split into a second muted track so you can bring some of it back in in a video editor.
Input: Video. Output: Video with isolated voic.

macOS only.

## Usage

* Right-click on the videos you want to split. Click "Open with" > "Dialogue Splitter"
* Alternatively double click the app and select videos in a dialog
* Output files are saved in a `splits` folder next to the original video

The output video has two audio tracks:
- **Track 1 (Voice)**: Isolated vocals only
- **Track 2 (Ambient)**: Original audio with vocals removed (Muted. Volume can be adjusted in video editing software)
